Notes

All concepts, types and functions documented are declared in the namespace boost::safe_float. Including policies declared in the namespace boost::safe_float::policy, and reporters declared in the namespace boost::safe_float::reporters.

B2 may output an error when detecting FENV, this is just part of the detection before compiling, not a real issue.

An operation producing a denormal result is considered an underflow by the policies provided by this library.

All results producing a NaN are considered invalid results by the policies provided by this library.
